# 💰 Smart Money Manager

Smart Money Manager is a **Full Stack Financial Management Web Application** that helps users track income, manage expenses, and visualize financial activity through an interactive dashboard.

The application allows users to easily monitor their financial transactions and understand their spending patterns.

---

## 🚀 Features

• User Authentication (Login / Signup)  
• Track Income  
• Track Expenses  
• Categorize Transactions  
• Filter Transactions  
• Financial Dashboard  
• Interactive Charts  
• Responsive UI
